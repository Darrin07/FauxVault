# FauxVault
A purposefully vulnerable web banking application built to demonstrate 
and document OWASP Top 10 attack vectors and their mitigations.
## Team
- Darrin Miller
- Nicholas Noochla-or
- Felicia Barrett

## Tech Stack
- Node.js / Express.js
- React
- PostgreSQL
- Docker / Docker Compose
- AWS EC2
## Getting Started
Clone the repo and run with Docker Compose:
1. git clone https://github.com/yourname/FauxVault.git
2. cd FauxVault
3. docker-compose up
## Disclaimer
This application is intentionally vulnerable and is for educational 
cybersecurity research only. Do not deploy on a public network.
